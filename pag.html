Git config

Este comando sirve para poder configurar el git-hub de tal modo que pueda iniciar de manera correcta todos los proyectos a realizarse

Git init

Este comando se utiliza para poder iniciar cualquier trabajo en el git-hub, con el cual damos por entendido que este es el primer paso para trabajar

Git remote

Este comando se utiliza para modificar el trabajo que se va a realizar consta de 2 comandos que son:

Git origin

Se utiliza para poder guardar el repositorio desde donde se puede copiar o clonar el mismo

Git

Este comando se utiliza para modificar o cambiar la direccion url del repositorio

Git add

Este comando sirve para poder ingresar un archivo a GITHUB, es decir poder guardalo y que este este agregado en el resto del trabajo

Git Status

Este comando se utiliza para poder ver el estado en el que se encuentra el trabajo que realizas, es decir si esta guardado o si tiene algun tipo de modificación

Git commit -m""

El commit sirve para poder poner una descripcion al trabajo que se esta realizando y asi tener un panorama mas claro de lo que se esta haciendo

Git push origin "branch"

El push sirve para poder subir el trabajo al git-hub web, el cual estara en tu reservorio para poder utilizarlo despues o para que otros puedan acceder a al mismo en caso de que sea publico
