GITHUB

Aprendiendo a usar git-hub


Es un portal creado para poder almacenar todo tipo de código creado por un usuario, estos pueden ser públicos o privados, dependiendo al trabajo que se realice

Comandos

Git clone

Este comando se utiliza para poder clonar cualquier tipo de trabajo siempre y cuando este sea de dominio publico, con él puedes clonar/copiar, el trabajo que desees sin ningun inconveniente

Git Status

Este comando se utiliza para poder ver el estado en el que se encuentra el trabajo que realizas, es decir si esta guardado o si tiene algun tipo de modificación

Git add

Este comando sirve para poder ingresar un archivo a GITHUB, es decir poder guardalo y que este este agregado en el resto del trabajo

Git Commit -m""

El commit sirve para poder poner una descripcion al trabajo que se esta realizando y asi tener un panorama mas claro de lo que se esta haciendo

Git push origin master

El push sirve para poder subir el trabajo al git-hub web, el cual estara en tu reservorio para poder utilizarlo despues o para que otros puedan acceder a al mismo en caso de que sea publico
